# Тест-кейсы платформы электронного обучения

**Версия:** 1.0  
**Дата:** июнь 2026  
**Стандарт оформления:** функциональное тестирование, формат «параметр → ожидаемый результат»  
**Окружение тестирования:** Windows 10/11, Chrome/Edge, Frontend :8000, Backend :5000, SQL Server

---

## 1. Обозначения

| Поле | Описание |
|------|----------|
| **ID** | Уникальный идентификатор тест-кейса |
| **Приоритет** | P1 — критический, P2 — высокий, P3 — средний |
| **Тип** | F — функциональный, N — негативный, UI — интерфейс, I — интеграционный, S — безопасность |
| **Предусловия** | Состояние системы до выполнения теста |
| **Шаги** | Последовательность действий тестировщика |
| **Параметры** | Входные данные |
| **Ожидаемый результат** | Корректное поведение системы |
