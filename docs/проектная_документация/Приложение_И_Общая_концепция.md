# Приложение И. Общая концепция

**Проект:** Адаптационный курс для сотрудников ритуальной компании  
**Версия:** 1.0  
**Дата:** июнь 2026

---

## Термины

| Термин | Определение |
|--------|-------------|
| **Рабочая область** | Основная часть макета приложения, предназначенная для вывода основного контента (карточки курсов, таблица рейтинга, формы) |
| **База данных** | Совокупность связанных таблиц SQL Server (`LearningPlatformDB`), организованная по реляционной модели |
| **LMS** | Learning Management System — платформа управления обучением |
| **Назначение (Assignment)** | Запись о том, что конкретному сотруднику назначен курс на определённый период |
| **Нативный курс** | HTML/JS курс без SCORM, взаимодействующий с API через `lms-bridge.js` |
| **Gating** | Блокировка навигации «Далее» до выполнения интерактивов на экране |
| **Эксперт** | Роль наставника: назначает курсы, формирует отчёты |
| **Обучающийся** | Роль сотрудника, проходящего назначенные курсы |

## Ключевые моменты

### Внешний вид

Внешний вид приложения лаконичен и не отвлекает от содержания. Графические элементы оформления не должны перекрывать рабочую область.

Элементы навигации (верхнее меню, кнопка ☰) располагаются **в одном месте на всех формах** — в шапке `.app-header`. Выпадающее меню аккаунта открывается под кнопкой ☰.

Внешний вид и алгоритм работы не должны вызывать затруднений у пользователя: понятные подписи кнопок («НАЧАТЬ КУРС», «Назначить курс», «Отчёт»), сообщения об ошибках на русском языке.

### Функционал

Основная задача — упростить процесс корпоративного обучения:

- эксперт назначает курс в несколько кликов;
- обучающийся видит только свои назначенные курсы;
- прогресс сохраняется автоматически при переходе между разделами;
- рейтинг и отчёты формируются из данных БД без ручного свода.

Второстепенная информация (модальные окна, выпадающие меню) загружается без перезагрузки всей страницы.

### Цветовая схема

**По брифу заказчика:** зелёный `#6E703E`, бежевый `#E7D8B6`, оранжевые акценты.

**В реализации LMS:** персиково-оранжевая палитра (`#FFCBA4`, `#FA8072`, `#D27D58`), серый фон `#F4F4F4`, тёмно-серый текст `#333333`. Курс «Безопасность» использует собственные стили в `course-player.css`.

### Визуальные образы

- шапка LMS с логотипом и навигацией по ролям;
- карточки курсов с обложкой и прогресс-баром;
- в курсе — фирменные SVG-иллюстрации из папки `картинки из курса/`.

## Идеология

Сайт отвечает современным требованиям подачи информации:

- крупные заголовки для быстрого визуального поиска;
- рабочая область без лишних рамок;
- адаптивная вёрстка курса для ПК, планшета и смартфона;
- масштаб 100% (рекомендация на странице «Моё обучение»).

Рекомендуется использование крупных шрифтов в заголовках разделов курса для навигации «по диагонали».

## Разделы приложения

### Главное меню по ролям

| Роль | Пункты меню |
|------|-------------|
| Обучающийся | Моё обучение, Рейтинг |
| Эксперт | Назначения, Рейтинг, Моё обучение |
| Администратор | Пользователи, Курсы |

### Меню аккаунта (☰)

| Роль | Пункты |
|------|--------|
| Обучающийся | Открыть профиль, Выйти |
| Эксперт | Личный кабинет, Выйти |
| Администратор | Открыть профиль, Выйти |

### Страница авторизации

Кнопка «Войти» на `login.html` переносит пользователя на стартовую страницу его роли после успешной аутентификации.

```
└─────────────────┘
```
