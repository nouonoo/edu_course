## Перенос в Word (проектная документация.docx)

1. Откройте каждый файл `.md` в редакторе с поддержкой Markdown.
2. Скопируйте текст в Word или конвертируйте через Pandoc:  
   `pandoc Приложение_А_Бриф.md -o Приложение_А.docx`
3. Диаграммы Mermaid экспортируйте в PNG (расширение Markdown Preview Mermaid, mermaid.live) и вставьте как рисунки с подписями.
4. Словарь данных (Приложение Ж) уже оформлен в табличном виде, совместимом с Word.
