# Приложение А. Бриф на разработку программного продукта

**Проект:** Адаптационный курс для сотрудников ритуальной компании  
**Версия:** 1.0  
**Дата:** июнь 2026

---

## 1. Общие сведения

### 1.1. Название проекта

Адаптационный курс для сотрудников ритуальной компании.

### 1.2. Какие операционные системы/устройства должен поддерживать сайт?

- Windows / macOS (десктопные браузеры: Chrome, Firefox, Edge);
- мобильные устройства (iOS / Android, адаптивная вёрстка курса и LMS);
- Smart TV (WebOS, Tizen) — базовая поддержка через веб-браузер.

### 1.3. Кто целевая аудитория сайта?

- пользователи всех возрастов;
- работники военно-ритуальной компании;
- эксперты (наставники, руководители подразделений);
- администраторы учебного отдела.

### 1.4. Какие ключевые функции должен выполнять сайт?

- прохождение электронных и SCORM-курсов;
- назначение курсов сотрудникам;
- просмотр рейтинга обучающихся;
- выгрузка отчётов о прохождении в формате Excel;
- управление пользователями и курсами (для администратора);
- ведение личного кабинета и загрузка фото профиля.

### 1.5. Нужна ли интеграция с другими сервисами? Если да, укажите какие.

Не нужна. Система работает автономно (Flask API + SQL Server + статический frontend).

### 1.6. Что ещё важно учесть при разработке?

- сайт должен быть быстрым (загрузка страниц LMS &lt; 1,5 сек при локальном развёртывании);
- удобный курс-плеер с последовательной навигацией, оглавлением и кнопкой «Закрыть курс»;
- разграничение доступа по ролям: администратор, эксперт, обучающийся;
- порог успешного прохождения курса — не менее 70%;
- интерфейс на русском языке.

## 2. Интерфейс и дизайн

### 2.1. Каким должен быть интерфейс?

- минималистичный;
- удобная, нативная навигация (верхнее меню по ролям, меню аккаунта ☰);
- единый стиль форм и карточек курсов.

### 2.2. Какие цвета использовать в дизайне?

**Заявленная цветовая схема заказчика:**

- основные: зелёный `#6E703E`, бежевый `#E7D8B6`;
- акценты: белый текст, оранжевые кнопки для действий и информационные блоки.

**Реализованная палитра LMS (frontend/style.css):**

- кнопки и акценты: `#FFCBA4`, `#FA8072`, `#D27D58`;
- фон страницы: `#F4F4F4`;
- основной текст: `#333333`.

### 2.3. Нужны ли готовые шаблоны для пользователей?

Да: шаблон выгрузки результатов тестирования (`course_report.xlsx`).

### 2.4. Какие языки должен поддерживать сайт?

Русский.

### 2.5. Потребуется ли обучение для администраторов?

Да: инструкция по добавлению контента и запуску системы (см. «Инструкция по запуску проекта.md», «Руководство пользователя»).
