# Приложение Ж. Словарь данных

**Проект:** Адаптационный курс для сотрудников ритуальной компании  
**Версия:** 1.0  
**Дата:** июнь 2026  
**СУБД:** Microsoft SQL Server

---

## Таблица 1. Поля таблицы Users

**Users** — учётные записи сотрудников

| KEY | FIELD NAME | DATA TYPE / FIELD SIZE | REQUIRED? | NOTES |
|-----|------------|------------------------|-----------|-------|
| PK | user_id | INTEGER | Y | IDENTITY(1,1) |
| | name | NVARCHAR(25) | Y | Имя |
| | surname | NVARCHAR(25) | Y | Фамилия |
| | patronymic | NVARCHAR(25) | N | Отчество |
| | birthday | DATE | N | Дата рождения |
| | phone | NVARCHAR(20) | N | Телефон |
| | status | NVARCHAR(20) | Y | active / inactive, DEFAULT active |
| FK | position_id | INTEGER | N | Ссылка на Position |
| | email | NVARCHAR(50) | Y | Уникальный логин |
| | password_hash | NVARCHAR(256) | Y | Пароль (учебный проект — открытый текст) |
| | photo | NVARCHAR(50) | N | Имя файла в uploads/photos |

## Справочные значения

### Roles.role_name

| role_id | role_name |
|---------|-----------|
| 4 | Администратор |
| 2 | Эксперт |
| 3 | Обучающийся |

### Assignments.status

| Значение | Описание |
|----------|----------|
| active | Назначение активно, курс в процессе |
| passed | Курс завершён с результатом ≥ 70% |
| failed | Курс завершён с результатом &lt; 70% |

### Section_id (курс «Безопасность»)

| section_id | Оцениваемый |
|------------|-------------|
| splash, navigation, intro | Нет |
| section-1 … section-6 | Да |
| conclusion | Нет |
